# twcdcnews
Taiwan CDC News (台灣衛福部疫管署新聞稿) PHP Class

例:
$cdc = new TWCDCNEWS(); // 查首頁資訊
或
$cdc = new TWCDCNEWS($today); // 指定日期搜尋新聞搞

$cdc->get_links(); 
// 回傳新聞稿的 link, 傳回陣列, 含 title 及 url 兩個欄位
